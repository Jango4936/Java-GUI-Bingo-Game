Bingo Game

Name: Chua Zhen Keat
Project: Bingo Game

===Description===
This project implements a simple Bingo game in Java. It allows 2 players to play Bingo by clicking on numbers and marking them on their bingo cards.

===Getting Started===
To run the Bingo Game, follow these steps:

1. Compile the Program: Open a terminal or command prompt, navigate to the directory containing the source code files, and compile them using the following command:

		javac Bingo_Test.java

2. Run the Program: After compiling, run the program by executing the following command:

		java Bingo_Test

Enjoy the Game: Once the program starts, follow the on-screen instructions to play the Bingo game. Have fun!
